# PomodoroBuddy v1.0.0

출시일: 2026-02-16

## 핵심 변경
- 타이머/휴식 플로우를 정비해 세션 종료 후 휴식 전환 동작을 안정화했습니다.
- `휴식 시간(분)`과 `휴식 시간 주기 ON/OFF` 설정을 추가했습니다.
- 세션 시작 후 초기화 전까지 카테고리/설정 편집 잠금을 적용했습니다.

## 카테고리/세션 개선
- 고정 카테고리(`휴식`, `공부하기`)를 보장하고 우선순위 정렬을 적용했습니다.
- 고정 카테고리는 수정/삭제가 불가능하도록 UI/로직을 보강했습니다.
- 휴식 세션 완료 시 직전 업무 카테고리/분으로 복귀하도록 개선했습니다.

## 통계/설정 UI 개선
- 통계 차트 Y축 시간을 `분/시간` 단위로 읽기 쉽게 표시하도록 개선했습니다.
- 설정 화면에 휴식 섹션을 추가하고 세션 중 편집 제한을 명확히 적용했습니다.
