# Custom Sound Files

Put your custom audio files in this folder.

Supported extensions:
- `.wav`
- `.m4a`
- `.mp3`
- `.aiff`

Required file names:
- Completion sounds:
  - `end-bell-01` (default)
  - `end-bell-02`
  - `end-bell-03`
- BGM sounds:
  - `bgm-rain`
  - `bgm-fire`

If a file is missing, that sound will be treated as silent.
