# PomodoroBuddy v1.0.2

## Fixes
- Fixed an app crash that occurred when changing sound settings in the Settings window.
- Hardened audio resource lookup so the app no longer terminates when a specific bundle path is unavailable.

## Build and Release Improvements
- Updated app bundle build script to support version/build-number parameters.
- Added a dedicated release zip build script for reproducible packaging.
